Sistem Operasi Linux

Linux merupakan salah satu contoh pengembangan perangkat lunak bebas dan sumber utama terbuka. Pada perangkat lunak yang bebas dan sumber terbuka lain pada umumnya, kode sumber Linux dapat dimodifikasi, digunakan dan didistribusikan secara bebas oleh siapa saja. Sistem Operasi Linux diciptakan oleh Benedict Torvalds pada tahun 1983. Sistem Operasi Linux mempunyai kelebihan dari Sistem Operasi lain, yaitu jauh lebih murah dibandingkan Sistem Operasi lain, merupakan Sistem Operasi yang memiliki code 32 bit, memilki tingkat keamanan yang cukup baik, tampilan grafik yang baik, mudah untuk dilakukan modifikasi terhadap source code, mendukung akses ke 33 macam sistem file yang berbeda, penggunaan Linux lebih terspesifikasi, software bawaan sudah terinstall dan dapat digunakan dengan mudah, cocok bagi yang hobi melakukan utak-atik komputer, cocok untuk dijadikan sebagai komputer server, spesifikasi hardware yang tidak terlalu besar, penggunaan software linux akan mencegah user dari pembajakan hak cipta. Sistem Operasi Linux ini juga mempunyai banyak versi yaitu yang disebut dengan Distro Linux.
