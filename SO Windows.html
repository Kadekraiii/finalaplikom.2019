Sistem Operasi Windows

Sistem Operasi merupakan sebuah sistem yang memiliki tugas untuk megendalikan operasi dasar dan akan memastikan sistem dalam komputer akan berjalan dengan normal dan semestinya. Sistem Operasi mengalami perkembangan yang dapat dibagi ke dalam lima generasi, yaitu dari generasi pertama pada tahun 1945-1955 sampai dengan sekarang. Salah satu yang temasuk dari Sistem Operasi adalah Sistem Operasi Windows, Sitem Operasi ini berkembang pesat itu ditandai dengan adanya beberapa versi dari sitem operasi Windows, diantaranya adalah Windows 1.0, Windows 2.0, Windows 3.0, Windows 95, Windows 98, Windows 2000, Windows Vista, Windows XP, Windows 7, Windows 8, dan Windows 10.
